APLICACIÓN WEB DESDE CERO – DIFUSIÓN DE EVENTOS UNIVERSITARIOS (TESIS)

Contexto

Crea una aplicación web completa desde cero como prototipo funcional para una tesis universitaria.
El objetivo del sistema es mejorar la difusión de eventos universitarios mediante la centralización de la información y una interfaz limpia, reduciendo el ruido digital.

A diferencia de un MVP mínimo, este prototipo sí incluye interacción social controlada, pero manteniendo el enfoque académico y evitando saturación visual.

🧠 OBJETIVO GENERAL DEL SISTEMA

Centralizar eventos académicos, culturales y deportivos

Permitir interacción entre estudiantes de forma moderada

Registrar métricas cuantificables para análisis estadístico

Mantener una interfaz clara, sin publicidad ni contenido externo

🧩 MÓDULOS PRINCIPALES

La aplicación tendrá tres vistas principales:

Vista Pública (sin login)

Vista Estudiante (con login)

Vista Administrativa (FIMAZ)
